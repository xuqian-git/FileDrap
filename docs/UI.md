# FileDrap UI 设计说明 (v0.1)

## 1. 设计原则
- 简洁：单窗口完成所有高频操作。
- 高效：左侧切目录，右侧直接拖文件。
- 低认知负担：保持 Finder 习惯（图标、路径、双击定位）。

## 2. 面板结构
- 顶部工具栏：
  - 品牌标题
  - Add Folder（主按钮）
  - Finder（打开当前目录）
  - 显隐文件开关
  - 排序切换按钮
- 主体双栏：
  - 左栏：文件夹列表（名称+路径）
  - 右栏：文件项列表（图标、文件名、路径）
- 底部动作：左栏下方提供删除当前文件夹按钮。

## 3. 视觉风格
- 使用系统语义色和原生控件，确保与 macOS 风格一致。
- 重点动作使用 `borderedProminent`，次要动作用 `bordered`/图标按钮。
- 空状态统一图标+标题+说明，避免空白页面。

## 4. 交互规则
- 单击文件夹即切换目录并刷新右侧列表。
- 双击文件项：在 Finder 定位。
- 拖拽文件项：输出文件 URL 到目标应用。
- 右键文件项：提供 Show in Finder。

## 5. 可用性细节
- 列表项路径单行截断，保证信息密度。
- 目录为空与无目录选中分开提示。
- 错误信息就地显示在文件列表顶部。
