# FileDrap PRD (v0.1)

## 1. 产品目标
- 在 macOS 菜单栏提供一个常驻的快速文件投递面板。
- 用户维护一组常用文件夹，能快速切换并从中拖拽文件到任意桌面应用。

## 2. 核心用户场景
- 设计师/运营需要频繁将素材拖到聊天、邮件、浏览器上传框。
- 开发者需要从多个工作目录快速拖文件到 IDE、终端、文档工具。
- 日常办公需要减少 Finder 多窗口切换成本。

## 3. 功能需求
### 3.1 必做（MVP）
- 菜单栏图标驻留，点击展开面板。
- 添加/移除常用文件夹。
- 展示文件夹列表（含路径），支持快速切换。
- 展示当前文件夹下文件列表。
- 支持将文件或子文件夹拖出到系统其他应用或桌面。
- 双击文件项在 Finder 定位。
- 用户配置本地持久化（下次启动自动恢复）。

### 3.2 应补充需求（你可能还没明确）
- 权限模型：
  - 通过用户选择文件夹获得访问权限。
  - 使用 security-scoped bookmark 持久化授权。
- 排序与过滤：
  - 名称升/降序。
  - 隐藏文件显示开关。
- 空状态与错误态：
  - 未添加文件夹、文件夹为空、读取失败都需要明确提示。
- 菜单栏交互细节：
  - 面板风格建议使用 window 弹窗，保证拖拽稳定性。
- 性能边界：
  - 超大目录先支持列表展示；后续可加分页/增量加载。

## 4. 非功能需求
- 启动快，常驻内存占用低。
- 操作路径短：添加后 2 次点击内可开始拖拽。
- 兼容 macOS 13+。

## 5. 后续迭代建议（v0.2+）
- 搜索过滤文件名。
- 文件夹置顶/拖动排序。
- 最近使用文件与固定文件。
- 全局快捷键呼出面板。
- 多窗口应用拖拽智能目标提示。
